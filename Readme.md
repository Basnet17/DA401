This is Pritam Basnet's DA 401 project. Here are the file structures:

1. Main file `yfin.py`: This file is used to get historical price data from stock market
using `yfinance` API which scrapes data.
   
2. 14Days.csv: This file has the stock market data -14 days to +14 days from the day the post was made in Reddit.

3. cleaned1.csv: This is the most cleaned file that contains Reddit statistics for 15 months.
